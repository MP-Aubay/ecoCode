The goal of this rule is to ban usage of "autoplay" attribute to make sure that media is not downloaded if user does not play it.

Noncompliant Code Example

<body>
  <audio autoplay src="/media/audio.mp3">  <!-- Noncompliant -->
    <a href="/media/audio.mp3">
      Download audio
    </a>
  </audio>
  <video autoplay src="/media/video.mp4">  <!-- Noncompliant -->
    <a href="/media/video.mp4">
      Download video
    </a>
  </video>
</body>

Compliant Solution

<body>
  <audio src="/media/audio.mp3">
    <a href="/media/audio.mp3">
      Download audio
    </a>
  </audio>
  <video src="/media/video.mp4">
    <a href="/media/video.mp4">
      Download video
    </a>
  </video>
</body>
